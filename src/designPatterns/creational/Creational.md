Very short comparison
Singleton → one shared instance
Factory Method → subclasses choose which object to create
Abstract Factory → create families of related objects
Builder → create complex objects step by step
Prototype → create objects by cloning existing ones
Easy way to remember
Need only one object? → Singleton
Need to choose one concrete type? → Factory Method
Need a whole matching family of objects? → Abstract Factory
Need to build something step by step? → Builder
Need to copy an existing object? → Prototype
